# **App Name**: Animuizu

## Core Features:

- App Intro: Landing page with logo, entrance animations and login/register options
- Join Game: Join random quiz matches and display questions/answers.
- Answer questions: Answer questions of multiple-choice format with limited time.
- Match Summary: See who won, review ranking after the game
- User Authentication: Allow the players to log in and log out
- Score Calculation: Calculate points of correct answer; add points based on answering time

## Style Guidelines:

- Dark background to evoke a late-night, focused viewing environment. In HSL, the hue should align with a vibrant color associated with anime and manga, such as purple. Use very low saturation (around 15%) and low brightness for a dark color scheme: #262329
- Primary color: A saturated purple, such as #A569BD (HSL: 292, 40%, 55%), as an energetic and fantastical element reminiscent of transformation sequences from magical girl anime.
- Accent color: #6E89B3, an analogous, desaturated, and lighter blue, HSL(220, 30%, 60%), for UI elements that require less emphasis but still need to stand out.
- Font: 'Belleza' (sans-serif) for titles, paired with 'Alegreya' (serif) for body, to bring an artistic vibe.
- Use a consistent set of icons (simple, flat style) to represent various actions and categories (e.g., user profile, settings, different anime genres).
- Use a grid-based layout to organize content. Responsive design for mobile and web.
- Smooth transitions and animations when questions appear, answers are selected, and results are displayed.
